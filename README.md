# mir-lapack
NDSLICE wrapper for LAPACK

## Required static libraries
 - `lapack` - FORTRAN 77 LAPACK API

## Wrapped API

 - gelsd
 - geqrf
 - gesdd
 - gesv
 - gesvd
 - getrf
 - getri
 - getrs
 - ormqr
 - potrf
 - potri
 - potrs
 - pptrf
 - pptri
 - spev
 - sptrf
 - sptri
 - sysv_rook
 - sytrf
 - sytrs2
 - tptri
 - trtri
 - unmqr
