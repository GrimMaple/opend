# mir-lapack
NDSLICE wrapper for LAPACK

## Required static libraries
 - `lapack` - FORTRAN 77 LAPACK API

## Wrapped API

 - gelsd
 - gesdd
 - gesv
 - gesvd
 - getrf
 - getri
 - spev
 - sytrf
 - geqrf
 - getrs
 - potrs
 - sytrs2
 - sptrf
 - sysv_rook
 - potrf
 - pptrf
 - sptri
 - potri
 - pptri
 - trtri
 - tptri
 - ormqr
 - unmqr
